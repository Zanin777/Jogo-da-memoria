# 🎮 MEMORIA — Arcade Memory Game

Jogo da memória com estética arcade/retro, feito com **HTML, CSS e JavaScript puro** (sem frameworks, sem dependências).

![status](https://img.shields.io/badge/status-jog%C3%A1vel-00F5D4) ![tech](https://img.shields.io/badge/stack-HTML%20%2B%20CSS%20%2B%20JS-FF2E97)

## ✨ Funcionalidades

- 🃏 Cartas com animação 3D de virada (CSS `transform` + `backface-visibility`)
- ⏱️ Timer e contador de movimentos em tempo real
- 🏆 Sistema de recorde por dificuldade
- 🎚️ Três níveis: Fácil (6 pares), Médio (8) e Difícil (12)
- 📱 Layout responsivo, funciona no celular
- ♿ Respeita `prefers-reduced-motion`
- 🎨 Visual arcade com efeito scanline e tipografia pixel art

## 🚀 Como rodar

Por ser 100% client-side, basta abrir o `index.html` no navegador.
Ou publique direto no **GitHub Pages**:

1. Suba os arquivos para um repositório
2. Vá em **Settings → Pages → Branch: main → Save**
3. Acesse `https://seu-usuario.github.io/nome-do-repo`
